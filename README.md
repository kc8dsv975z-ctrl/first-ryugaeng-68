## 2. ps 명령어
`ps` 명령어는 **현재 실행 중인 프로세스의 목록을 출력하는 명령어**이다.  
`top`과 달리 정적인(멈춰있는) 정보만 보여줌.

### 자주 쓰는 옵션

| 옵션 | 의미 |
|------|------|
| `ps -e` | 모든 프로세스 표시 |
| `ps -f` | 프로세스의 상세 정보 표시 |
| `ps aux` | CPU/메모리 사용량 포함 모든 프로세스 표시 |
